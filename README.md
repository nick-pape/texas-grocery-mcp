# Texas Grocery MCP

MCP server for HEB grocery store integration with AI agents.

## Installation

```bash
pip install texas-grocery-mcp
```

## Development

```bash
# Clone the repository
git clone https://github.com/your-username/texas-grocery-mcp.git
cd texas-grocery-mcp

# Create virtual environment
python3 -m venv .venv
source .venv/bin/activate

# Install with dev dependencies
pip install -e ".[dev]"
```

## Usage

```bash
texas-grocery-mcp
```
